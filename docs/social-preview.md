# Social Preview Direction

## Goal
Create a GitHub social preview image for **Agent Operating Model Kit** that feels:
- open-source
- clean and premium
- governance-first
- framework-agnostic
- more like an operating model / systems kit than an AI mascot product

## Visual Direction

### Core idea
Show the repository as a **layered operating model kit** rather than a chatbot or a coding tool.

### Should communicate
- doctrine
- runtime principles
- governance gates
- recurring review
- assetization / compounding structure

### Should avoid
- mascots
- anime characters
- robots or humanoids
- too much text
- framework-specific branding
- chaotic cyberpunk noise
- startup-sales-poster energy

## Composition
- Dark background
- Clean grid or systems-diagram feel
- Layered cards / panels / modules
- Thin lines or faint connectors
- One or two accent colors only
- Strong center composition suitable for GitHub Open Graph preview

## Text on Image
Keep text minimal because social preview images should survive low-resolution previews.

Recommended text:
- Title: `Agent Operating Model Kit`
- Subtitle: `Governance overlay for AI organizations`

If the model struggles with subtitle rendering, keep only the main title.

## Color / Mood
- charcoal / near-black background
- cool gray structure
- cyan or electric blue accent
- optional soft purple secondary accent

## Style words
- minimal
- modular
- systems-oriented
- strategic
- crisp
- premium open-source tooling

## Intended Use
- GitHub social preview image
- README / repo sharing card
- possible future docs hero image

## Generation Notes
- Prefer abstract system modules over literal icons
- Keep the image legible at small preview sizes
- Avoid dense labels; structure should carry meaning visually
