# Weekly Operating Review Prompt

Run a weekly review for a single-customer micro AI organization.

Review the last week through four company-level KPIs:
1. customer value output
2. token efficiency
3. cycle time
4. assetization rate

For each KPI, answer:
- what went well
- what was weak or wasteful
- one concrete structural improvement

Then answer:
- Which work created real downstream value?
- Which work consumed budget without enough value?
- Which repeated tasks should become assets?
- Which doctrine principles were most frequently upheld?
- Which doctrine principles were most frequently violated?

End with:
- one thing to stop
- one thing to continue
- one thing to institutionalize
